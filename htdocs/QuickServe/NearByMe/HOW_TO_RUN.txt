❗ TROUBLESHOOTING
─────────────────

Problem: "This site can't be reached"
Solution: 
- Check if Apache is running in XAMPP
- Verify URL: http://localhost/NearByMe/ (with capital N and M)

Problem: "Connection failed"
Solution:
- Check if MySQL is running in XAMPP
- Go to XAMPP Control Panel and start MySQL

Problem: "Table doesn't exist"
Solution:
- Run database setup: http://localhost/NearByMe/database/init-database.php
- Refresh the page

Problem: "Page not found / 404 error"
Solution:
- Verify path is correct: C:\xampp\htdocs\NearByMe\
- Check folder name spelling (capital N and M)

Problem: "CSS not loading / ugly page"
Solution:
- Clear browser cache (Ctrl + Shift + Delete)
- Hard refresh page (Ctrl + F5)
- Check if style.css exists in assets/css/
