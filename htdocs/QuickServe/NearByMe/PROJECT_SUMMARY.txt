SETUP INSTRUCTIONS:
──────────────────

STEP 1: Install XAMPP
   → Download from: https://www.apachefriends.org/
   → Install and start Apache + MySQL

STEP 2: Copy Files
   → Copy NearByMe folder to: C:\xampp\htdocs\

STEP 3: Setup Database
   → Open: http://localhost/NearByMe/database/init-database.php
   → This creates database, tables, and sample data automatically

STEP 4: Access Application
   → Open: http://localhost/NearByMe/
   → Login with demo credentials above

HOW TO RUN:
──────────

1. Make sure XAMPP is installed and running
2. Ensure MySQL and Apache are started in XAMPP Control Panel
3. Copy project to: C:\xampp\htdocs\NearByMe\
4. Run database setup: http://localhost/NearByMe/database/init-database.php
5. Access application: http://localhost/NearByMe/
6. Login with demo credentials provided above

TROUBLESHOOTING:
───────────────

❌ "Connection failed"
   ✅ Solution: Start MySQL in XAMPP

❌ "Table doesn't exist"
   ✅ Solution: Run init-database.php script

❌ "Page not found"
   ✅ Solution: Check path is C:\xampp\htdocs\NearByMe\

❌ "CSS not loading"
   ✅ Solution: Clear browser cache (Ctrl+Shift+Del)
